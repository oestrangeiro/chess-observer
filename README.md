<h1>Chess Observer</h1>
<p>Crawler em PHP que coleta estatísticas dos perfis dos meus amigos no chess.com e salva localmente em um arquivo JSON</p>

<h3>Para coletar as métricas dos seus amigos</h3>
<p>Abra o arquivo <code>index.php</code> e modifique o array <code>$listOfPlayers</code> e rode <code>php index.php</code> </p>
<p>As métricas ficarão salvas no arquivo "friends.json"</p>
